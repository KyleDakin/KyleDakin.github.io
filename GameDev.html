About

Game made in Unity with C#. Uses Python through a port to control the game. Computer vision done with OpenCV. Uses haar cascades, aruco marker tracking and colour picking. Inspired by Captain Toad Treasure Tracker, made a game that required the player to move the camera, using face tracking, to explore a map to uncover hidden areas needed to complete the level
