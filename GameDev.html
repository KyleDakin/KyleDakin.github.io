About

The aim of this module was to design a game that implements VR/AR technologies as part of the gameplay. To do this, I decided upon a puzzle platformer game inspired by Captain Toad's Treasure Tracker.
The gameplay was coded in Unity in C# and the level was made using Unity's new ProBuilder tool. The controls for the game were split into two parts. The player movement was controlled using an app developed in Unity and a Python programme that ran colour picking and ArUco marker tracking algortihms, while the camera controls were done entirely using facial tracking algorithms also done in the Python programme.
The game requires rotating and raising the camera to explore the map and discover hidden areas, where treasures and key to progress through the game. The project uses a Python program through a port to control the both the camera and the player in game using various computer vision algorithms such as ArUco Marker tracking. Computer vision done with OpenCV. Uses haar cascades, aruco marker tracking and colour picking. Inspired by Captain Toad Treasure Tracker, made a game that required the player to move the camera, using face tracking, to explore a map to uncover hidden areas needed to complete the level
